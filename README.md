# schedule-app

スケジュール管理アプリケーション

## インストール

### Windows

1. [リリースページ](https://github.com/Egssy620/schedule-app/releases)から最新版の `schedule-app-setup-X.X.X.exe` をダウンロードします
2. ダウンロードしたexeファイルを実行し、インストールウィザードに従ってインストールを完了します

**⚠️ セキュリティ警告について**

初回インストール時にセキュリティ警告が表示される場合があります。その場合は以下の手順で進めてください：

1. 警告ダイアログで「詳細」ボタンをクリック
2. 「インストールを続行する」ボタンが表示されるのでクリック
3. インストールが続行されます
